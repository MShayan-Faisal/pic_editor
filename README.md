🎨 Pic Editor – Simple Online Photo Editor

Pic Editor is a lightweight, browser-based photo editing web app built using HTML, CSS, and JavaScript. It allows users to upload images and apply various filters, adjustments, and effects instantly — without needing heavy desktop software.

The goal was to create a fast, easy-to-use image editor that feels modern, responsive, and intuitive — perfect for quick edits, previews, or creative experimentation directly in the browser.

About The Project

Pic Editor was designed to make image editing simple and accessible. Instead of relying on complex programs, users can drag and drop an image, adjust its brightness, contrast, saturation, blur, and apply filters — all within seconds.

The project focuses on both functionality and design. The interface is clean and minimal, ensuring the tools are easy to understand while maintaining an aesthetically pleasing layout.

It demonstrates how powerful the web platform can be when combining the flexibility of JavaScript with the precision of CSS.

Technologies Used

HTML5 – for layout and structure
CSS3 – for interface styling and responsive design
JavaScript – for image processing, filter effects, and interactivity

Key Features

Upload and preview images instantly
Apply filters like brightness, contrast, grayscale, sepia, and blur
Reset or download edited images easily
Responsive layout for all screen sizes
User-friendly interface with real-time updates

How It Works

The user uploads an image using the file input or drag-and-drop area.

JavaScript reads the image and renders it onto the screen using the <canvas> element.

Filter sliders and buttons allow real-time adjustments to properties like brightness, contrast, blur, and saturation.

The user can reset the filters or download the edited image with a single click.

This workflow keeps everything smooth and efficient, using native browser APIs without external libraries.
